# Password Reset - Flow

## Feature Summary

- **Password Reset**
  - **Flow**: Implement secure password reset functionality that allows users to reset their passwords via email verification in two steps: request and verification.
  - **Step 1**: [Password Reset Request](step1.md) - User submits email to initiate password reset process
  - **Step 2**: [Password Reset Verification](step2.md) - User completes password reset using verification link and new password
